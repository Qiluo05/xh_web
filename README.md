# xh-website

## 介绍
> 主要面向社团管理的门户网站  

该项目主要面向校内社团的管理 辅以门户网站常见的媒体传播功能 实现社团管理、刺激社团活力  
现潦草实现管理功能 更多界面还待不断优化
